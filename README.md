The SPCC-Plan skill is a custom Claude skill stored as a structured set of Markdown files — a main SKILL.md that defines the workflow, and three reference files covering interview questions, a section-by-section plan template, and a comprehensive gap-checking rulebook. 

When triggered, Claude reads these files at runtime and follows a six-phase process: it first extracts any information already provided, then interviews the user to collect site-specific data across nine topic groups (facility identity, oil storage inventory, containment, drainage, operations, inspections, security, training, and spill history). From those answers, it automatically determines the correct regulatory tier under 40 CFR Part 112 — Tier I Qualified Facility, Tier II, or a Full PE-Certified Plan — and drafts all thirteen sections of the plan using site-specific data, proper CFR citations, and pre-populated defaults for known facility types like data centers. 

Throughout drafting, it applies a comprehensive gap-checking rulebook to flag every missing or non-compliant element inline with color-coded severity levels (🔴 Critical, 🟡 Important, 🔵 Recommended), then hands off to the DOCX skill to produce a fully formatted, print-ready Word document complete with cover page, table of contents, headers, footers, signature blocks, and appendix templates.
